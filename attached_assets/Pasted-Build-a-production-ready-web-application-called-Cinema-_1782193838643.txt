Build a production-ready web application called **Cinema AI Translator**.

## Objective

Create a web application that provides near real-time translation of movie audio into a user-selected language.

Primary use case:

A censorship board member or reviewer sits in a cinema wearing headphones. The application listens to the movie audio through the device microphone, automatically detects the spoken language, translates the dialogue into the selected target language, generates translated speech, and plays it through connected headphones.

The application should work entirely through a web browser.

### Audio Processing Workflow

When user presses START:

1. Request microphone permission.
2. Begin continuous audio capture.
3. Record audio in 5-second chunks.
4. Upload chunk to backend.
5. Send audio to OpenAI speech recognition.
6. Automatically detect spoken language.
7. Convert speech to text.
8. Translate transcript into selected target language.
9. Generate translated speech using OpenAI TTS.
10. Stream generated speech back to browser.
11. Play translated audio through browser output device.
12. Continue processing next chunk automatically.

---

### Continuous Translation Loop

The system should continuously process:

Audio Chunk
→ Transcript
→ Translation
→ Speech
→ Playback

until user presses STOP.
